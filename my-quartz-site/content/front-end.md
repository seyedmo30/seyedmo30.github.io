# front end

## nuxt

برای گیت هاب پیج میشه روی یک برنچ کد ها رو گذاشت بالا ، و روی یک برنچ دیگه ، تنها بیلد شده ها رو گذاشت بالا

```

نصب

npm install

تست و توسعه

npm run dev

بیلد مخصوص گیت هاب پیج

npx nuxt build --preset github_pages

جنریت مخصوص گیت هاب پیج

npx nuxt generate --preset github_pages

پوش روی برنچ مخصوص پروداکشن

npx gh-pages --dotfiles -d .output/public
```
